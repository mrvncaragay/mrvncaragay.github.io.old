Gareta E-Comerce (gareta-e4fdd.firebaseapp.com)
Gareta is an e-comerce for both Web and Mobile platform. This application is built React, Redux/Saga, Firebase, Google Auth, Amazon S3, and Material UI. End users are able to view view products, add to cart, and check out using Stripe Payments service. This project is uploaded to Github (github.com/mrvncaragay/Gareta)

TECHNOLOGIES:

React Redux / Saga Firebase / DB / Hosting Amazon s3 Material UI JavaScript HTML/CSS
